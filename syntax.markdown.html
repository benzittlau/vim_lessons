22 The `:highlight` command will allow you to configure different 'highlight groups'
23 with a chosen color.
24 
25 If you are in a cterm, try:
26 # Type `:highlight Normal ctermbg=black ctermfg=white guibg=black guifg=white`
27 
28 Pretty, isn't it!
29 
30 The highlight allows you to apply styling to a `highlight group`.  Assuming we are all
31 dealing with color terminals or graphical vims, these are the available commands that
32 can be passed to highlight:
33 
34           GUI                        CTERM
35     gui={attr-list}              cterm={attr-list}
36     font={font-name}
37     guifg={color-name}           ctermfg={color-name}
38     guibg={color-name}           ctermbg={color-name}
39 
40 Any number of the above commands can be used in a single call to 'highlight'.  The
41 commands that don't apply to your given vim type will be ignored.  Note that our example
42 above used commands for both the GUI and for CTERM, so it will work for both.
43 
44 The 'Normal' means we're setting colors for the normal highlight group.
45 Let's try setting colors for 'Visual' highlight group!
46 
47 Try:
48 # Type `:highlight Visual ctermbg=red ctermfg=blue guibg=red guifg=blue`

54 So what do 'gui' and 'cterm' do?  These can apply additional non-color
55 stylings to a highlight group.  Let's have some fun:
56 
57 # Type `:highlight htmlH2 gui=underline cterm=bold`
58 ## If you are using a gui vim this text is now underlined!
59 ## If you are using a cterm vim this text is now bold!
60 
61 There are a few other options you can play with here...
62 # Type `:highlight htmlH3 gui=standout,reverse cterm=standout,reverse`
